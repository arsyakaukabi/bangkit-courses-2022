Kategori : Knowledge Check - Pemrograman Berorientasi Objek

Terdapat sebuah class Kalkulator seperti berikut:

class Kalkulator:
 
    def __init__(self, nilai=0):
        self.nilai = nilai
 
    def tambah_angka(self, angka1, angka2):
        self.nilai = angka1 + angka2
        if self.nilai > 9:             
            print('kalkulator sederhana melebihi batas angka: {}'.format(self.nilai))
        return self.nilai

Kemudian kita ingin membuat sebuah class yang memiliki fitur pengurangan dengan mewarisi atribut dan metode dari class Kalkulator tanpa mengubah class Kalkulator itu sendiri, apakah kode di bawah ini sudah sesuai dengan kondisi yang diinginkan di atas?

class KalkulatorKurang:
 
    def kali_angka(self, angka1, angka2):
        self.nilai = angka2 - angka1
        return self.nilai
